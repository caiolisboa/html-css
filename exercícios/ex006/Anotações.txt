CTRL + SHIFT + P // ENVELOPAR 

<strong> // negrito
<em> // italico ou enfase

<abbr> // abreviaçoes
<code> // deixa os espacos uniforme
<pre> // pré-formatação ( contabiliza os espaços dados)
<del> // texto excluso (cortado)
<ins> // sublinhado
<big>  e <small> // texto grande e pequeno
<mark> // marcar
